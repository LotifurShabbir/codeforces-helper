# Codeforces Helper 🚀

Codeforces Helper is an advanced AI skill designed to act as a **Competitive Programming Coach**. It doesn't just give you the code; it teaches you the hidden intuition, handles edge cases, and writes optimal C++ solutions.

## 👑 What It Does

When you paste a Codeforces problem link or description, this skill provides:
1. **📚 Prerequisites:** Topics you need to know before solving.
2. **💡 Intuition & Observation:** The "Aha!" moment or hidden math.
3. **⏱ Constraint Analysis:** Determining the exact Time/Space complexity needed.
4. **🚧 Edge Cases & WA Traps:** Common mistakes and tricky inputs.
5. **💻 Clean C++ Code:** Beginner-friendly logic with Fast I/O setup.
6. **🔍 Dry Run:** Step-by-step trace table.
7. **🎯 Up-Solving:** Next topics to study and similar problems to practice.

## 📁 Skill Files

- `SKILL.md`: Metadata and high-level triggers.
- `AGENTS.md`: The core brain! Contains detailed behavior rules, C++ standards, and the strict answer format.

## 🛠️ How to Use
1. Download this repository as a `.zip` file.
2. Go to your AI platform (like Claude with custom skills enabled).
3. Create a new skill and upload/paste the files.
4. Type `/codeforces-helper` followed by a problem link.
5. Level up your CP rating!
